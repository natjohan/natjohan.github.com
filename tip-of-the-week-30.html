Tip of the week 30

Gnome EncFS Manager

As I switched on ElementaryOs (which is a really great distrib) few months ago, I just discovered a new tool to mount my EncFS folders. Actually CryptKeeper didn't work properly under Elementary (due to some bugs with the tray system). So, I find a tool called Gnome EncFS Manager, and it works great. Here is the way to install it under Elementary (or Ubuntu)

natjohan:~$ sudo add-apt-repository ppa:gencfsm && sudo apt-get update && sudo apt-get install gnome-encfs-manager

Then you can easily manage(mount/umount/create) your encrypted folders


Note : you can install this tool under many other distro (Fedora, Debian, OpenSuse and Arch)
