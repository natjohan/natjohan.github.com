Tip of the week 34

In a previous ToTW, I showed you how to manage and install package under Sublime Text. Here is a useful markdown plugin that I use to edit my markdown files : https://github.com/SublimeText-Markdown/MarkdownEditing .

Note : If you don't know the markdown format just goes here to learn more, in few words from my point of view : markdown is a text format that allow you to focus only on your content rather than how it looks (like the rst or lateX format)

To install it under Sublim Text through Package Control just type ctrl+shift+p then Package Control : install package, enter key and type Markdown editing, enter key again, and that's all the package install should be done.

Now you can chose the MarkdownEditing synthax under your Sublime Text editor.
